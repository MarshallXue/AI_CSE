岸岸通 UI Moodboard

目标方向：清爽备考桌面 + 沉浸式闪卡复盘。下面这些参考用于讨论视觉风格和组件结构，不直接照搬商业素材。

设计判断：首页和错题库要有秩序，新闻和词汇要有阅读感，复盘页要有沉浸感。

三套可组合风格

A. 清爽公考工具风

适合今日页、错题库、复盘首页。强调可信、效率和模块秩序。

沉稳蓝白 任务面板 弱数据化

B. 阅读杂志感

适合新闻详情、时政精读和词汇例句。强调纸面阅读、留白和层级。

轻纸感 正文舒适 关键词高亮

C. 沉浸闪卡学习风

适合错题、词汇、时政考点复盘。强调单卡聚焦和少干扰反馈。

单卡聚焦 轻翻卡 三档反馈

官方 App 截图参考

粉笔

工具感 / 公考模块

看模块组织、题目解析层级、练习结果表达。不要照搬课程营销密度。

不背单词

沉浸卡片 / 语境

看大卡片、真实语境、重点高亮。不要照搬壁纸化和英语学习情绪包装。

Quizlet

闪卡流 / 简洁反馈

看单卡复习、进度表达和少干扰模式。适合复盘页参考。

AnkiMobile

专业闪卡 / 进度

看复习评分逻辑和卡组管理。不要照搬复杂设置和工程化视觉。

扇贝

每日任务 / 复习规划

看今日任务、记忆规划、学习完成反馈。适合复盘首页参考。

墨墨背单词

抗遗忘 / 复习节奏

看长期记忆规划表达。不要照搬英语词库的复杂学习路径。
