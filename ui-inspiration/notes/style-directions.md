# 候选 UI 风格方向

## A. 清爽公考工具风

关键词：

- 可信
- 清楚
- 高效
- 任务面板
- 弱数据化

适合：

- 今日页
- 错题库
- 复盘首页

视觉特征：

- 白底和浅灰蓝背景。
- 沉稳蓝作为主色。
- 卡片紧凑但不拥挤。
- 数据用小卡片或紧凑指标表达。
- 状态标签轻量化。

优点：

- 公考用户熟悉。
- 功能心智清楚。
- 长期使用不累。

风险：

- 容易像传统题库 App。
- 如果缺少阅读和闪卡质感，会显得普通。

## B. 沉浸闪卡学习风

关键词：

- 单卡聚焦
- 低干扰
- 轻仪式感
- 语境学习
- 翻卡反馈

适合：

- 错题闪卡
- 词汇闪卡
- 时政考点闪卡

视觉特征：

- 中央大卡片。
- 顶部进度极简。
- 正反面内容分层。
- 底部固定三档反馈。
- 轻微翻卡动效。

优点：

- 产品记忆点强。
- 最贴合艾宾浩斯复习。
- 适合碎片时间。

风险：

- 如果全 App 都做成沉浸卡片，会降低工具效率。
- 大图和强氛围不适合公考错题。

## C. 阅读杂志感 + 复盘工具风

关键词：

- 纸面阅读
- 新闻精读
- 考点层级
- 温和留白
- 轻组件

适合：

- 新闻详情
- AI 精读
- 时政考点卡生成结果
- 词汇例句阅读

视觉特征：

- 更舒适的正文行距。
- 标题、摘要、关键词、考点分层。
- 使用分割线和留白，不是堆卡片。
- 重要关键词加粗或浅色高亮。

优点：

- 能提升今日时政的质感。
- 和公考材料阅读场景匹配。

风险：

- 做太“杂志”会降低任务感。
- Awwwards 式动效不能直接套进学习工具。

## 推荐组合

不要三选一，而是按页面组合：

- 今日页：A + C
- 新闻详情：C
- 错题库：A
- 复盘首页：A + B
- 闪卡页：B
- 词汇页：B + C
- 导入页：A

一句话：

> 首页和错题库有秩序，新闻和词汇有阅读感，复盘页有沉浸感。
