# 岸岸通 UI 灵感资料夹

这个资料夹用于收集岸岸通 UI 改版前的参考材料和调研结论。

## 文件结构

- `screenshots/appstore/`
  - 官方 App Store 截图参考，包含粉笔、不背单词、Quizlet、AnkiMobile、扇贝、墨墨背单词。
  - 这些截图仅用于内部 moodboard 和设计讨论，不作为产品资产直接使用。
- `notes/research-summary.md`
  - 多智能体调研汇总。
- `notes/component-shortlist.md`
  - 可参考的组件模式和来源链接。
- `notes/style-directions.md`
  - 三套候选 UI 风格方向。
- `notes/source-links.md`
  - 外部灵感来源链接清单。

## 当前推荐方向

综合调研后，最适合岸岸通的方向是：

> 清爽备考桌面 + 沉浸式闪卡复盘

设计关键词：

- 清爽可信
- 备考效率
- 纸面阅读感
- 轻量数据化
- 单卡聚焦
- 克制动效

## 使用原则

- 借鉴信息层级、组件结构、交互模式和视觉气质。
- 不照搬具体 UI、截图、插画或商业素材。
- Dribbble、Behance、Awwwards、ArtStation 等作品以链接和观察为主，不下载作品图作为可用资产。
- App Store 截图仅作为竞品/参考分析材料。
