# 公考错题解析技能

当用户给出一道公考题、OCR 识别结果、错题记录或答案解析请求时，使用这个技能。

## 工作流程

1. 先判断题型：常识、行测、申论素材、资料分析、判断推理、言语理解。
2. 提取题干关键词、用户答案、正确答案、考点。
3. 给出清晰的解题路径。
4. 解释为什么正确选项正确。
5. 分析用户错因，例如概念混淆、审题过快、关键词记忆不牢、计算路径错误。
6. 给出下次复盘建议。
7. 如适合，生成 1-3 道同类强化题。

## 输出格式

优先用中文，结构如下：

- 题型判断
- 核心考点
- 解题步骤
- 正确答案说明
- 错因分析
- 复盘建议
- 同类题提醒
