# 岸岸通错题与闪卡复盘设计

## 背景

岸岸通当前聚焦两个核心板块：

1. 今日板块：分享公考时政新闻和今日词汇。
2. 错题复盘板块：整理错题，并按间隔重复进行闪卡复习。

本设计回应用户反馈：

> 希望做一个按照艾宾浩斯遗忘曲线来复习复盘的错题整理工具，复习模式按照闪卡模式，未掌握的继续推送复习，掌握的不推送，并且线上 App 错题和纸质错题能整理到同一个 App。

第一版目标不是做大而全学习平台，而是打通：

> 今日学习 / 用户导入 -> 自动整理 -> 进入对应复习池 -> 闪卡复盘 -> 根据状态排期 -> 掌握后退出日常推送

## 产品结构

### 今日板块

今日板块负责输入新内容。

包含：

- 今日新闻
- 今日词汇
- 新闻详情
- 新闻配套练习

进入复习池规则：

- 用户点开新闻详情后，系统生成该新闻的时政考点挖空卡，进入时政考点复习池，默认状态为 `需巩固`。
- 用户在新闻配套练习中答错后，错题自动进入错题库和错题复盘池，默认状态为 `需巩固`。
- 用户点开今日词汇组后，该组词汇进入词汇复习池，默认状态为 `需巩固`。

### 错题库

错题库负责管理错题。

来源：

- 用户自主导入错题。
- 今日新闻配套练习答错后自动收录。

错题库按公考模块组织：

- 常识时政
- 资料分析
- 言语理解
- 判断推理
- 数量关系
- 申论

筛选维度：

- 全部
- 未掌握
- 需巩固
- 已掌握
- 今日待复习
- 逾期未复习
- 来源

错题列表展示弱化状态标签，不抢题目标题。错题详情展示完整状态、下次复习时间、复习次数和最近一次反馈。

### 复盘板块

复盘板块是闪卡复习中心，不是错题库列表。

首页入口：

- 开始今日全部复盘
- 错题闪卡
- 词汇闪卡
- 时政考点闪卡

用户可以在设置中配置今日全部复盘的类型和顺序。默认使用上一次设置，用户可以调整。

## 闪卡形态

### 错题闪卡

正面展示：

- 题干
- 选项或材料
- 来源：自主导入 / 今日练习
- 模块
- 知识点标签

交互：

- 选择题和判断题支持先点选答案。
- 主观题和复杂资料分析题支持点 `我已作答`。
- 始终保留 `直接看解析` 按钮。

背面展示：

- 正确答案
- 用户答案或上次错误答案
- AI 解析
- 错因分析
- 知识点
- 下次复习建议
- 反馈按钮：`未掌握 / 需巩固 / 已掌握`

如果用户直接看解析，系统记录本次为跳过作答，但仍允许用户手动选择状态反馈。

### 词汇闪卡

词汇来源使用现有词库 `apps/frontend/src/app/data/vocabulary-entries.json`。当前词库包含成语和实词，并已按词汇组、释义、例句等字段组织。今日词汇推送数量由已有前端设置控制。

用户点开词汇组后，该组词汇进入词汇复习池。

每个词生成两类卡：

1. 词义卡
   - 正面：词语。
   - 背面：释义、类型、易混提醒、例句。

2. 语境卡
   - 正面：例句挖空。
   - 背面：完整例句、词语解释、使用场景。

示例：

正面：

> 中华文明 ____，在历史长河中不断焕发新的生命力。

背面：

> 中华文明 **源远流长**，在历史长河中不断焕发新的生命力。

### 时政考点闪卡

时政考点卡采用挖空卡，不采用泛泛问答卡。

生成规则：

- 用户点开新闻详情后生成。
- 按文章重要程度决定数量。
- 普通新闻生成 1-2 张。
- 重要新闻生成 2-4 张。
- 热点政策或高频考点生成 4-6 张。
- 内容价值低时不强行生成。

卡片类型：

- 基础卡：记事实、关键词、主体、时间、数字、核心表述。
- 进阶卡：记政策逻辑、考法、申论表达、易错点。

Agent 负责根据规则自动区分基础卡和进阶卡，用户可以编辑卡片内容和难度标签。

示例：

正面：

> “藏粮于地、藏粮于技”强调把粮食安全基础放在 ____ 和 ____ 上。

背面：

> “藏粮于地、藏粮于技”强调把粮食安全基础放在 **耕地保护** 和 **农业科技** 上。
>
> 关键词：粮食安全、耕地红线、科技赋能、综合生产能力。

## 状态与复习排期

所有复习对象统一使用三个状态：

- 未掌握
- 需巩固
- 已掌握

适用对象：

- 错题
- 词汇
- 时政考点

### 默认进入状态

- 词汇组被用户点开后，进入词汇复习池，默认 `需巩固`。
- 新闻详情被用户点开后，时政考点卡进入复习池，默认 `需巩固`。
- 自主导入错题先进入错题库，不自动进入复习池。
- 用户点击自主导入错题的 `加入复盘` 后，进入错题复习池，默认 `需巩固`。
- 新闻配套练习答错后，进入错题库和错题复习池，默认 `需巩固`。

### 排期规则

第一版采用固定节奏加动态调整。

默认节奏：

`当天 / 次日 -> 3 天 -> 7 天 -> 14 天 -> 30 天`

反馈后的规则：

- `未掌握`
  - 默认次日再推。
  - 如果开启当天强化，10-30 分钟后再次进入今日复盘。
  - 状态更新为 `未掌握`。

- `需巩固`
  - 1 天后再推。
  - 状态更新为 `需巩固`。

- `已掌握`
  - 进入下一个长间隔。
  - 间隔可按 3 天、7 天、14 天、30 天递进。
  - 状态更新为 `已掌握`。

### 退出日常推送

- 连续 3 次跨天标记 `已掌握` 后，退出日常复习队列。
- 内容仍保留在库里。
- 考前复习和专项复习时可以召回。
- 召回后如果被标记为 `未掌握` 或 `需巩固`，重新进入日常复习队列。

### 复盘设置

设置放在复盘设置中，不打扰主流程。

包含：

- 复盘顺序：错题闪卡、词汇闪卡、时政考点闪卡。
- 当天强化：默认关闭。
- 每日复盘上限：例如 20 / 40 / 60 / 不限。
- 考试日期：用于考前召回。

## 错题导入与整理

第一版支持三种自主导入：

- 拍照：纸质试卷、练习册、手写错题。
- 图片或截图上传：其他学习 App、网页、电子资料中的错题截图。
- PDF 上传：其他平台导出的真题、练习、讲义、试卷。

### PDF 处理

PDF 第一版采用按页预览和选择页导入。

流程：

1. 用户上传 PDF。
2. 系统生成每页预览。
3. 用户选择要整理的页面。
4. 支持 `全选`。
5. 选中页面按图片页处理。
6. 系统执行 OCR、切题、识别、解析、分类。

### OCR 方案

接入百度智能云 OCR API。

识别后自动生成：

- 题干
- 选项
- 用户答案 / 正确答案，能识别则识别
- AI 解析
- 模块分类
- 知识点
- 错因
- 来源
- 掌握状态

全自动整理是主流程，但识别结果都允许用户编辑。

可编辑字段：

- 题干
- 选项
- 答案
- 解析
- 模块
- 知识点
- 错因
- 状态标签

### 自主错题进入复盘

- 自主导入错题默认只进入错题库。
- 用户点击 `加入复盘` 后进入错题复习池。
- 加入后默认状态为 `需巩固`。

## 页面与入口

### 今日页面

保留今日新闻和今日词汇切换。

新增或明确行为：

- 点开新闻详情后生成时政考点挖空卡。
- 新闻配套练习答错后进入错题库和复习池。
- 点开词汇组后进入词汇复习池。

### 错题库页面

顶部：

- 搜索
- 导入入口

快捷导入：

- 拍照
- 上传图片
- 上传 PDF

主体：

- 按模块显示文件夹。
- 模块详情页显示错题列表。
- 每道题展示弱化状态标签。
- 支持状态、任务和来源筛选。

### 复盘页面

复盘首页突出闪卡入口。

展示：

- 今日待复盘总数
- 预计时间
- 开始今日全部复盘
- 错题闪卡入口
- 词汇闪卡入口
- 时政考点闪卡入口

每个入口展示：

- 今日到期数量
- 逾期数量
- 完成进度

点击开始今日全部复盘：

- 默认按用户设置顺序开始。
- 用户可以调整本次复盘类型和顺序。
- 按类别分段进入闪卡流。

### 闪卡页

统一结构：

- 顶部进度，例如 3/20。
- 中间卡片主体。
- 正面展示题目、词语或挖空。
- 背面展示答案、解析或解释。
- 底部反馈：`未掌握 / 需巩固 / 已掌握`。

## 第一版范围

### 包含

- 今日新闻和今日词汇作为内容入口。
- 今日词汇点开后进入词汇复习池。
- 新闻详情点开后生成时政考点挖空卡并进入复习池。
- 新闻配套练习答错后进入错题库和错题复习池，默认 `需巩固`。
- 用户自主导入错题：拍照、图片/截图上传、PDF 上传。
- PDF 按页预览，选择页导入，支持全选。
- 接入百度智能云 OCR API。
- 自动切题、识别、解析、分类。
- 用户可编辑识别结果。
- 错题库按公考模块组织。
- 复盘页做成三类闪卡中心。
- 统一状态：`未掌握 / 需巩固 / 已掌握`。
- 固定节奏加动态调整的复习排期。
- 连续 3 次已掌握后退出日常推送。
- 考前和专项复习可召回。
- 复盘设置：顺序、当天强化、每日上限、考试日期。

### 不包含

- 从其他 App 自动登录同步错题。
- 系统分享导入。
- 爬虫抓取第三方题库。
- 自部署 PaddleOCR。
- 完整 FSRS 或 Anki 高级算法。
- 社区题库或公开题库。
- 多人班级或教师端。
- 复杂学习报告。
- AI 对话中的错题保存。

## 成功标准

第一版成功标准：

- 用户能从今日新闻和今日词汇自然产生复习内容。
- 用户能通过拍照、截图或 PDF 把外部错题整理进错题库。
- 用户能在复盘页清楚进入错题、词汇、时政三类闪卡。
- 每张卡都能通过 `未掌握 / 需巩固 / 已掌握` 更新状态和下次复习时间。
- 已掌握内容不会长期打扰日常复习，但能在考前或专项复习中召回。
- 复盘页和错题库在功能心智上明显区分：错题库负责管理，复盘页负责训练。
